# Task 1: Data Structures Implementation (C++)

## Objective
Implement core data structures from scratch using C++.

Implemented:
1. Singly Linked List
2. Binary Search Tree (BST)

---

## Environment Notes

- Operating System: Windows 11
- IDE: Visual Studio Code
- Language: C++
- Compiler: g++ (MinGW GCC)

To check compiler version:

```bash
g++ --version
```

---

## Build and Run Commands

### Linked List

Compile:

```bash
g++ linked_list.cpp -o linkedlist
```

Run:

```bash
./linkedlist
```

Windows PowerShell:

```powershell
.\linkedlist.exe
```

---

### Binary Search Tree

Compile:

```bash
g++ bst.cpp -o bst
```

Run:

```bash
./bst
```

Windows PowerShell:

```powershell
.\bst.exe
```

## Data Structures Implemented

### 1. Singly Linked List

Operations:
- Insert
- Search
- Delete
- Display

Time Complexity:

| Operation | Complexity |
|-----------|------------|
| Insert | O(1) |
| Search | O(n) |
| Delete | O(n) |

---

### 2. Binary Search Tree (BST)

Operations:
- Insert
- Search
- Delete
- Inorder Traversal

Time Complexity (Average Case):

| Operation | Complexity |
|-----------|------------|
| Insert | O(log n) |
| Search | O(log n) |
| Delete | O(log n) |
